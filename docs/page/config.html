配置 ​

DZ Player 的参数遵循 w3c 规范，,同时支持原生 video 的参数，播放器在实例化时的配置选项，开发者可以根据自己的需求进行选择性地配置,以下是 Dz Player 的参数列表：

容器 container ​

  • 类型: HTMLElement
  • 默认值: null
  • 说明: 必需

播放器实例化的时候需要明确DOM的占位，video将挂载到该DOM下，播放器的尺寸与占位DOM一致，挂载目标元素

js
new DzPlayer({
  container: document.getElementById('player'),
  ...
})

视频源 url ​

  • 类型: String | Object[]
  • 默认值: ''
  • 说明: 必需

URL为数组形式时，会寻找参数default:true 进行播放，如果没有配置默认为true的，则默认播放第一个视频。此参数可设置清晰度配置

js
//url为string时：
new DzPlayer({
  container:document.getElementById('player'),
  url: '//abc.com/**/*.mp4',
  ...
});
//url为数组时：
new DzPlayer({
  container:document.getElementById('player'),
  url: [
        {
            label:'HD',
            value:"//abc.com/**/*.mp4",
            default:true 
        },
  ],
  ...
});

警告

容器和视频源是播放器关键的配置项，缺一不可。

视频类型 type ​

  • 类型: String
  • 默认值: auto
  • 说明: 可选

可选值：'auto' | 'normal' | 'hls' | 'flv' | 'dash'

播放器宽度 width ​

  • 类型: String
  • 默认值: 100%
  • 说明: 可选

播放器高度 height ​

  • 类型: String
  • 默认值: 100%
  • 说明: 可选

封面 poster ​

  • 类型: String
  • 默认值: null
  • 说明: 可选

预加载 preload ​

  • 类型: String
  • 默认值: none
  • 说明: 可选

可选值：'auto' | 'metadata' | 'none'

是否静音 muted ​

  • 类型: Boolean
  • 默认值: false
  • 说明: 可选

音量 volume ​

  • 类型: Number
  • 默认值: 0.7
  • 说明: 可选

可选值范围：0.0 - 1.0

隔空播放 airplay ​

  • 类型: Boolean
  • 默认值: false
  • 说明: 可选

是否循环播放 loop ​

  • 类型: Boolean
  • 默认值: false
  • 说明: 可选

是否自动播放 autoplay ​

  • 类型: Boolean
  • 默认值: false
  • 说明: 可选

提示：autoplay为true时，浏览器会自动播放，如果浏览器不支持自动播放，则会触发autoplay_error事件。

播放速率 playbackRate ​

  • 类型: Number
  • 默认值: 1
  • 说明: 可选

是否显示控制条 controls ​

  • 类型: Boolean
  • 默认值: true
  • 说明: 可选

控制条配置 controlOptions ​

  • 类型: Object
  • 默认值: null
  • 说明: 可选
js
new DzPlayer({
  container: document.getElementById('player'),
  //...
  controlOptions: {
    //可选，是否显示原生控制条 true ｜ false
    nativeControls:false,
    //可选，是否显示中间大播放暂停按钮 true ｜ false
    bigPlayControl:true,
    //可选，是否显示刷新按钮,直播模式下不显示 true ｜ false
    flushControl: false,
    //可选，是否显示进度条 true ｜ false
    seekBar:true,
    //可选，是否显示播放时间 true ｜ false
    playTime:true,
    //可选，是否显示音量控制 true ｜ false
    volumeControl:true,
    //可选，是否显示全屏按钮 true ｜ false
    fullScreenControl:true,
    //可选，是否手动挂载 true ｜ false
    manualMount:false,
    //可选，控制条挂载目标
    mountTarget:document.getElementById('xxx'),
    //可选，自定义控制条样式
    controlStyle:"width:200px;height:100px;top:50px;right:50px;left:auto;"
  }
})

控制器的挂载目标 controlTarget ​

  • 类型: HTMLElement
  • 默认值: null
  • 说明: 可选

默认情况下，播放器的控件会挂载到播放器容器上，可以通过该参数来指定挂载目标。

跨域资源共享 crossOrigin ​

  • 类型: String
  • 默认值: null
  • 说明: 可选

提示：可选值：'anonymous' | 'use-credentials' crossOrigin为anonymous时，浏览器会自动跨域请求，如果浏览器不支持跨域请求，则会触发crossorigin_error事件。

显示水印配置 waterMark ​

  • 类型: Object
  • 默认值: null
  • 说明: 可选
js
new DzPlayer({
  container: document.getElementById('player'),
  //...
  waterMark: {
    //是否显示水印 true ｜ false
    show:false,
    //水印url地址
    url:"//abc.com/**/*.png",
    //水印样式
    style:"width:200px;height:100px;top:50px;right:50px;left:auto;"
  }
})

视频片段的开始时间 clipStart ​

  • 类型: Number
  • 默认值: 0
  • 说明: 可选

视频片段的结束时间 clipEnd ​

  • 类型: Number
  • 默认值: 视频结束时间
  • 说明: 可选

handleVideoEndByOuter ​

  • 类型: Boolean
  • 默认值: false
  • 说明: 可选

提示：是否由外部控制视频结束 ,用于视频片段播放 ,为 true 时，视频播放结束时不会触发 video end 事件

播放器自定义样式 customStyle ​

  • 类型: String
  • 默认值: ''
  • 说明: 可选

Video自定义样式 videoStyle ​

  • 类型: String
  • 默认值: ''
  • 说明: 可选

倍速列表 speedList ​

  • 类型: Array
  • 默认值: null
  • 说明: 可选
js
new DzPlayer({
  container: document.getElementById('player'),
  //...
  speedList: [
    {
      label:'1倍数',
      value:1.0
    },
    {
      label:'2.0X',
      value:1.5
    },
    {
      label:'3.0X',
      value:3.0
    },
  ]
})

自定义控制栏按钮 toolBars ​

  • 类型: Object
  • 默认值: null
  • 说明: 可选

开发者可通过该参数自定义控制栏按钮

js
new DzPlayer({
  container: document.getElementById('player'),
  //...
  toolBars: {
    //名称
    live:{
      title:"hello",//标题
      icon:"fas fa-yin-yang",//可选，图标，可配合font-awesome使用
      text:`LIVE`,//可选，文本，支持HTML标签
      position:'left', //可选，显示位置，left | right
      style:"font-size:16px;",//可选，自定义样式
      click:(player) => { //可选，回调点击事件
        console.log(player);
        player.play();
      }
    },
    //...
  }
})
