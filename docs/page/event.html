属性和API ​

DZ Player 的参数遵循 w3c 规范，,同时支持原生 video 的参数，播放器在实例化时的配置选项，开发者可以根据自己的需求进行选择性地配置,以下是 Dz Player 的参数列表：

属性 ​

原生 video ​

js
const dp = new DzPlayer({
  container: document.getElementById('player'),
  //...
})
console.log(dp.video);

事件列表 EventsList ​

Video事件 ​

事件名说明
audioprocess当音频处理程序处理缓冲区时触发
canplay当浏览器能够开始播放音频/视频时触发
canplaythrough当浏览器可在不因缓冲而停顿的情况下进行播放时触发
durationchange当音频/视频的时长已更改时触发
completeOfflineAudioContext 渲染完成触发
emptied当目前的播放为空时触发
ended当目前的播放已到达末尾时触发
loadeddata当浏览器已加载视频的元数据时触发
loadedmetadata当浏览器已加载视频的元数据时触发
pause当视频播放已暂停时触发
play当视频开始播放时触发
playing当视频开始播放时触发
progress在浏览器加载资源时周期性触发
ratechange当视频的播放速度已更改时触发
seeked当用户已移动/跳跃到视频中的新位置时触发
seeking当用户开始移动/跳跃到视频中的新位置时触发
stalled当浏览器尝试获取媒体数据，但数据不可用时触发
suspend当浏览器刻意不获取媒体数据时触发
timeupdatecurrentTime 属性指定的时间发生变化
volumechange当音量已更改时触发
waiting由于暂时缺少数据，播放已停止触发
error在发生错误时触发
abort在视频终止加载时触发
loadstart在浏览器开始加载媒体数据时触发
mozaudioavailable当音频数据可用时触发
