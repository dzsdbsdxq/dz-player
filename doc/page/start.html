快速开始 ​

安装 Dz Player ​

sh
# 使用 npm:
npm i dz-player
# 使用 Yarn:
yarn add dz-player
# 使用 Pnpm:
pnpm add dz-player

基本使用 ​

1. 在页面提供占位 DOM ​

html
<div id="player"></div>
1

2. 实例化 ​

使用npm包

js
import DzPlayer from 'dz-player'

const tp = new DzPlayer({
  container: document.getElementById('player'),
  src: 'demo.mp4',
})
